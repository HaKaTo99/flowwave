# Visi WaveforAI

Automation mudah, gratis, komunitas-driven. Hybrid: Offline for privacy, online for AI/collaboration. Seperti Wikipedia untuk workflows.

Filosofi:
- **Open Source (MIT)**: Bebas digunakan dan dimodifikasi.
- **Gratis + Donasi Sukarela**: Model keberlanjutan berbasis komunitas.
- **Offline-First, Online Optional**: Fokus pada privasi dan aksesibilitas, dengan kemampuan cloud untuk fitur advanced.
- **Aksesibel & Ekstensibel**: Dirancang untuk semua orang, dari pemula hingga developer.
